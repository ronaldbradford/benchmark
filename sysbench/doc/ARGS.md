# `sysbench` Command Line Arguments

`sysbench` accepts the following arguments with version `1.0.20`.

```
--config-file=
--db-debug
--db-driver=
--db-ps-mode=
--debug
--events=
--forced-shutdown=
--help
--histogram
--luajit-cmd=
--mysql-compression
--mysql-compression-algorithms=
--mysql-db=
--mysql-debug
--mysql-dry-run
--mysql-host=
--mysql-ignore-errors=
--mysql-password=
--mysql-port=
--mysql-socket=
--mysql-ssl-ca=
--mysql-ssl-cert=
--mysql-ssl-cipher=
--mysql-ssl-key=
--mysql-ssl=
--mysql-user=
--percentile=
--rand-pareto-h=
--rand-seed=
--rand-type=
--rand-zipfian-exp=
--rate=
--report-checkpoints=
--report-interval=
--thread-init-timeout=
--thread-stack-size=
--threads=
--time=
--validate
--verbosity=
--version
--warmup-time=
```
